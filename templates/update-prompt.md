# AI评测网站更新Prompt模板

## 使用方法
将以下内容发送给AI助手（如Cursor、ChatGPT等），按照格式填写相关信息即可完成网站更新。

---

## 更新指令

```
请帮我更新AI评测功能网站，相关信息如下：

### 1. 项目位置
/home/taojin/Documents/AI-Rating

### 2. 本次更新主题
[填写本次要更新的主题，如：GPT-4最新评测动态 / 多模态模型评测趋势 / AI安全评测等]

### 3. 信息来源（可选）
- 主要来源：[URL或平台名称]
- 补充来源：[其他相关URL]
- 关键信息点：[列出3-5个核心要点]

### 4. 更新类型（多选）
□ 添加新的评测专家
□ 添加新的评测工具
□ 添加新的评测流程文档
□ 添加新的评测报告
□ 更新3D图谱数据
□ 其他：[说明]

### 5. 具体要求
[描述你希望AI如何处理这些信息，如：
- 总结关键趋势
- 提取专家观点
- 对比分析
- 预测未来发展等]

### 6. 附加说明
[任何其他要求或特殊指示]
```

## 完整Prompt示例

```
请帮我更新AI评测功能网站，相关信息如下：

### 1. 项目位置
/home/taojin/Documents/AI-Rating

### 2. 本次更新主题
GPT-4最新评测动态与Claude模型对比分析

### 3. 信息来源
- 主要来源：https://openai.com/blog, https://www.anthropic.com
- 补充来源：Hugging Face, Papers with Code
- 关键信息点：
  1. GPT-4在MMLU基准上的最新得分
  2. Claude 3与GPT-4的能力对比
  3. 两者的安全对齐策略差异
  4. 多模态能力评测结果
  5. 未来发展趋势预测

### 4. 更新类型
☑ 添加新的评测报告（GPT-4 vs Claude对比）
☑ 添加新的评测工具介绍
☑ 更新3D图谱数据（添加新节点）
☑ 添加新的专家观点

### 5. 具体要求
1. 生成一份详细的对比评测报告
2. 总结两位专家对模型发展的观点
3. 分析AI评测领域的未来趋势
4. 更新3D图谱，添加新的节点和关系

### 6. 附加说明
使用中文输出，保持与现有文档风格一致
```

---

## AI助手应执行的操作

1. **信息收集**：访问提供的URL，提取关键信息
2. **内容生成**：
   - 生成新的Markdown文档（评测专家/工具/流程/报告）
   - 更新`assets/data/file-index.json`添加新文件索引
   - 更新`assets/data/3d-graph-data.json`添加新节点
3. **趋势分析**：分析领域发展趋势
4. **保存文件**：将生成的内容保存到对应目录
5. **报告完成**：告知用户生成了哪些文件

---

## 文件路径规范

- 评测专家：`docs/expert-profiles/[名字].md`
- 评测工具：`docs/tool-descriptions/[工具名].md`
- 评测流程：`docs/process-documents/[流程名].md`
- 评测报告：`docs/conclusion-reports/[报告名].md`
- 其他文档：`docs/other-related/[文件名].md`
